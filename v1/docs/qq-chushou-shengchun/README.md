# QQ 主题「触手生春」设计摘录

本目录收录第三方 QQ 皮肤「**触手生春**」的视觉语言归纳，用于在网页或工具中还原**春日 pastel 渐变 + 磨砂玻璃 + 薰衣草紫强调**的聊天界面气质。说明文档并非腾讯官方设计规范，颜色与尺寸为近似归纳。

## 文件

| 文件 | 说明 |
|------|------|
| `DESIGN.md` | 完整设计说明：色彩、字体、布局、组件、效果与 token |
| `preview.html` | 浅色主题静态预览（三栏会话界面 + 设计 token 展示） |
| `preview-dark.html` | 深色主题静态预览（同结构，暗色玻璃与光斑） |

可将 [DESIGN.md](./DESIGN.md) 作为 AI 或设计与前端协作时的单一事实来源，生成与「触手生春」风格一致的 QQ 式桌面聊天 UI。

## 预览说明

在浏览器中直接打开 `preview.html` 或 `preview-dark.html` 即可查看。页面包含：

- 模拟 **左侧导航 + 会话列表 + 聊天主区** 布局；
- **己方/对方气泡**、**系统时间条**、**输入栏与发送按钮**；
- **色板与圆角、模糊参数**摘录，便于对照 `DESIGN.md`。
