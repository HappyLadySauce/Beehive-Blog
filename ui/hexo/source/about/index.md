## 主题定位

`happyladysacue` 是一套面向内容站点的 Hexo 自定义主题，目标不是复刻 `AnZhiYu` 的全部交互，而是保留它强辨识度的首页气质，并用更轻的实现完成核心体验。

## 技术思路

- 使用 `Hexo 8 + EJS` 作为模板层，降低维护复杂度。
- 使用 `CSS Variables` 建立设计系统，统一暗黑模式、色板和组件形态。
- 使用 `Pretext` 为精选卡片与摘要文本提供高度预估，减少布局跳动。
- 使用 `Beehive` API 契约完成登录、注册、导航用户态与退出登录。

## 设计原则

- 内容优先：正文可读性高于炫技动效。
- 结构清晰：首页负责品牌感，列表页负责扫描效率，文章页负责沉浸阅读。
- 渐进增强：无 JavaScript 时仍可正常阅读与导航。
