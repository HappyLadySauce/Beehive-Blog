# Design System Inspiration — QQ 主题「触手生春」

本文档描述腾讯 QQ 第三方/自定义皮肤「触手生春」在视觉与交互上的可复现设计特征，供实现桌面端聊天界面、主题预览页或设计还原时使用。该主题并非 QQ 官方公开设计规范文档，颜色与尺寸为基于实机界面的归纳与近似值。

## 1. Visual Theme & Atmosphere

「触手生春」以**春日、轻盈、通透**为关键词：大面积**粉—黄—绿—蓝**方向的柔和渐变作为环境色，叠加**磨砂玻璃（Glass / Acrylic）**与低对比度的**线描或水彩感花卉底纹**，使界面像「春景透过薄雾」。浅色模式下整体偏亮、空气感强；深色模式下以**炭灰 / 深紫灰**为底，保留同款渐变光斑与花卉纹样，但饱和度与明度降低，形成**高级暗色玻璃**质感。

**关键特征：**

- **玻璃拟态**：面板使用 `rgba` 半透明填充 + `backdrop-filter: blur()`，边框极弱或仅用 1px 高光描边。
- **大圆角**：窗口外层、会话选中条、气泡与按钮普遍使用 **12px–20px** 级圆角，头像保持正圆。
- **主强调色**：**薰衣草紫**（约 `#A594D3` / `#9B92D1`）用于侧栏选中态、己方消息气泡、主按钮「发送」等。
- **层次**：背景渐变与装饰层固定或铺满；聊天区与列表为浮在之上的半透明层；消息气泡再叠一层，形成清晰纵深。
- **图标与线型**：细线、单色或弱渐变图标，避免厚重描边。

## 3. Typography Rules

### Font Family

- **中文界面**：`system-ui, "PingFang SC", "Microsoft YaHei UI", "Noto Sans SC", sans-serif`
- **英文与数字**：与中文栈合并，保持与 QQ 客户端接近的**现代无衬线**观感。
- **字重**：标题与昵称多用 **500–600**；正文 **400**；弱化信息 **400 + 降低透明度**。

### Hierarchy（桌面端参考）

| 角色 | 字号 | 字重 | 行高 | 说明 |
|------|------|------|------|------|
| 窗口/群标题 | 14–15px | 600 | 1.35 | 顶栏标题 |
| 会话昵称 | 14px | 500 | 1.35 | 会话列表 |
| 消息正文 | 14px | 400 | 1.5 | 气泡内 |
| 预览/次要 | 12–13px | 400 | 1.4 | 列表副行、时间 |
| 系统时间/提示 | 12px | 400 | 1.3 | 居中灰条 |

### Principles

- 中文场景下**行高略大于英文**，保证多行气泡可读性。
- 深色模式**避免纯黑字叠在浅紫气泡上**：使用 `--text-on-accent` 或微调气泡明度保证对比度（建议 WCAG AA 以上）。

## 4. Layout & Structure

### 典型布局 A：三栏主界面（消息列表模式）

适用于「窄导航 + 会话列表 + 聊天主区」的桌面布局。

| 区域 | 宽度策略 | 说明 |
|------|----------|------|
| 左侧导航栏 | 固定约 `52–64px` | 图标竖排，一项选中时用薰衣草色块或高亮条 |
| 会话列表 | 固定约 `240–280px` | 每项：圆形头像 + 昵称一行 + 摘要一行；选中项圆角背景 |
| 聊天主区 | `flex: 1` | 顶栏（头像+名称+工具）、消息滚动区、底部输入区 |

### 典型布局 B：群聊 + 右侧资料栏

适用于截图中的宽屏群聊：左侧为**主聊天区**，右侧为**群公告 + 成员列表**（固定约 `22–28%` 宽度或 `min(320px, 28vw)`）。

- 主聊天区与侧栏之间为**细分割线**或仅靠背景差分隔。
- 底部：**工具图标行** + **输入框** + **关闭 / 发送**（发送为主强调色填充）。

## 5. Component Stylings

### 窗口容器

- **外框圆角**：约 `16–24px`。
- **边框**：`1px solid rgba(255,255,255,0.25)`（浅）或 `rgba(255,255,255,0.06)`（深）。
- **阴影**：柔和、大扩散、低透明度，避免生硬黑色投影。

### 导航项 / 会话项

- **导航图标**：约 `40×40px` 可点区域，选中背景为圆角矩形或圆形底。
- **会话行高度**：约 `64–72px`；头像约 `44–48px`。
- **选中态**：浅主题为淡紫半透明条；深主题为 `rgba(255,255,255,0.08)` 或浅紫透明条。

### 消息气泡

- **圆角**：`14–18px`；小尾巴可省略（QQ 新版多为圆角矩形）。
- **对方**：左对齐，白或浅灰半透明气泡，文字 `--text-primary`。
- **己方**：右对齐，薰衣草紫填充，文字浅色。
- **系统/时间**：水平居中窄条，小字号，灰背景或无边框仅文字。

### 聊天区背景装饰

- **花卉线稿**：使用低透明度 PNG/SVG 平铺或 `background-size: cover` 置于消息层之下。
- **抽象色块**：大半径 `blur(60–120px)` 的径向渐变圆形，模拟花瓣光斑。

### 输入区域

- **工具条**：emoji、截图、文件、图片等线性图标，颜色 `--text-secondary`。
- **输入框**：透明或极浅玻璃底，底边框可选 `1px solid rgba(0,0,0,0.06)`。
- **按钮**：「关闭」为中性灰边或浅填充；「发送」为 `--accent-lavender` 填充、白字。

### 链接预览卡片（富媒体）

- 白或近白底、小圆角、内边距统一；标题与摘要层级分明。

## 6. Effects & Motion

- **磨砂**：`backdrop-filter: saturate(140%) blur(20px)`（值可按性能调整）。
- **过渡**：`transition: background-color 0.2s ease, transform 0.15s ease` 用于 Hover。
- **滚动**：消息区 `overflow-y: auto`；列表区独立滚动，避免整页抖动。

## 7. Accessibility & Performance

- 玻璃效果在**低端设备**可降级为纯色半透明，无 `backdrop-filter`。
- 动效尊重 `prefers-reduced-motion`。
- 所有可点击区域建议 **≥ 40×40px** 触摸/点击目标。

## 8. Assets Notes

- **花卉底纹**：建议使用**矢量或高分辨率**素材，导出为 SVG 或 WebP；透明度控制在 **6%–15%** 以免干扰阅读。
- **品牌**：若做第三方主题预览，避免使用腾讯注册商标作为官方背书表述。
