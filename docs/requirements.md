# 个人博客系统需求文档

## 1. 项目概述

### 1.1 项目名称
Beehive-Blog 个人博客系统

### 1.2 项目背景
本项目是一个基于 Go 语言开发的个人博客系统，旨在提供一个功能完善、易于管理的博客平台，支持文章发布、标签管理、评论互动、附件管理等核心功能。

### 1.3 技术栈
- **后端语言**: Go
- **数据库**: PostgreSQL
- **缓存**: Redis
- **API 文档**: Swagger
- **部署**: Docker

---

## 2. 用户角色定义

| 角色 | 角色标识 | 权限级别 | 说明 |
|------|----------|----------|------|
| 访客 | `guest` | 0 | 访客用户，可浏览内容和评论 |
| 普通用户 | `user` | 1 | 普通用户，可浏览内容和发表评论 |
| 管理员 | `admin` | 2 | 系统管理者，拥有全部权限 |

## 3. 功能需求

### 3.1 用户认证模块

#### 3.1.1 用户注册
- **功能描述**: 新用户注册账号
- **输入**: 用户名、密码、邮箱（可选）
- **输出**: 注册成功/失败信息
- **业务规则**:
  - 用户名必填，长度 3-20 字符
  - 密码长度 6-32 字符，需加密存储
  - 邮箱格式校验（如提供邮箱，需符合标准邮箱格式）
  - 需要邮箱验证，验证链接发送至用户邮箱，用户需点击链接激活账号

#### 3.1.2 用户登录
- **功能描述**: 用户身份验证
- **输入**: 用户名/邮箱、密码
- **输出**: JWT Token、用户信息
- **业务规则**:
  - 支持用户名或邮箱登录
  - 密码错误次数限制（5次/小时）
  - Token 有效期 7x24 小时

---

### 3.2 文章管理模块

#### 3.2.1 查看文章
- **功能描述**: 浏览文章列表和详情
- **权限**: 所有用户
- **功能点**:
  - 文章列表分页展示
  - 支持按分类、标签筛选
  - 支持关键字搜索
  - 文章详情展示（标题、内容、作者、发布时间、标签、评论数等）

#### 3.2.2 增加文章（管理员）
- **功能描述**: 发布新文章
- **权限**: 仅管理员
- **输入**: 标题、内容、分类（可选）、标签（可选）、封面图（可选）、状态
- **业务规则**:
  - 标题必填，长度 1-200 字符
  - 内容支持 Markdown 格式
  - 支持草稿和发布两种状态

#### 3.2.3 编辑文章（管理员）
- **功能描述**: 修改已有文章
- **权限**: 仅管理员
- **业务规则**:
  - 保留文章历史版本（可选）
  - 更新时间自动记录

#### 3.2.4 删除文章（管理员）
- **功能描述**: 删除文章
- **权限**: 仅管理员
- **业务规则**:
  - 软删除，保留数据可恢复
  - 删除时关联删除文章-标签关系

#### 3.2.5 文章分类管理（管理员）
- **功能描述**: 管理文章分类
- **权限**: 仅管理员
- **功能点**:
  - 创建分类（名称、描述、排序）
  - 编辑分类
  - 删除分类（需检查是否有关联文章）

#### 3.2.6 文章状态管理（管理员）
- **功能描述**: 设置文章状态
- **权限**: 仅管理员
- **状态定义**:
  - `draft`: 草稿
  - `published`: 已发布
  - `archived`: 已归档

---

### 3.3 标签管理模块

#### 3.3.1 查看标签
- **功能描述**: 浏览所有标签
- **权限**: 所有用户
- **功能点**:
  - 标签列表展示
  - 显示每个标签关联的文章数量
  - 点击标签查看关联文章

#### 3.3.2 增加标签（管理员）
- **功能描述**: 创建新标签
- **权限**: 仅管理员
- **输入**: 标签名称、颜色（可选）
- **业务规则**:
  - 标签名称唯一，长度 1-50 字符

#### 3.3.3 编辑标签（管理员）
- **功能描述**: 修改标签信息
- **权限**: 仅管理员

#### 3.3.4 删除标签（管理员）
- **功能描述**: 删除标签
- **权限**: 仅管理员
- **业务规则**:
  - 删除时自动解除与文章的关联关系

---

### 3.4 评论管理模块

#### 3.4.1 查看评论
- **功能描述**: 查看文章下的评论
- **权限**: 所有用户
- **功能点**:
  - 按文章查看评论列表
  - 支持评论分页
  - 支持评论回复（嵌套展示）

#### 3.4.2 发表评论
- **功能描述**: 对文章发表评论
- **权限**: 登录用户
- **输入**: 文章ID、评论内容、父评论ID（回复时）
- **业务规则**:
  - 评论内容长度 1-2000 字符
  - 支持回复其他评论
  - 新评论默认状态为 `pending`

#### 3.4.3 删除评论（管理员）
- **功能描述**: 删除不当评论
- **权限**: 仅管理员
- **业务规则**:
  - 软删除
  - 删除父评论时，子评论一并处理

#### 3.4.4 评论状态管理（管理员）
- **功能描述**: 审核评论状态
- **权限**: 仅管理员
- **状态定义**:
  - `pending`: 待审核
  - `approved`: 已通过
  - `rejected`: 已拒绝
  - `spam`: 垃圾评论

---

### 3.5 附件管理模块

#### 3.5.1 查看附件
- **功能描述**: 浏览上传的文件
- **权限**: 仅管理员
- **功能点**:
  - 附件列表展示
  - 支持按类型、时间筛选
  - 显示文件大小、上传时间

#### 3.5.2 上传附件
- **功能描述**: 上传图片/文件
- **权限**: 仅管理员
- **支持类型**:
  - 图片: jpg, jpeg, png, gif, webp
  - 文档: pdf, doc, docx, xls, xlsx, ppt, pptx
  - 其他: zip, rar, tar.gz
- **业务规则**:
  - 单文件最大 1GB
  - 自动生成缩略图（图片）
  - 文件重命名避免冲突

#### 3.5.3 删除附件
- **功能描述**: 删除附件
- **权限**: 仅管理员
- **业务规则**:
  - 物理删除文件
  - 删除数据库记录

#### 3.5.4 附件分类
- **功能描述**: 管理附件分类
- **权限**: 仅管理员
- **分类类型**:
  - `image`: 图片
  - `document`: 文档
  - `other`: 其他

---

### 3.6 用户管理模块（管理员）

#### 3.6.1 查看用户
- **功能描述**: 浏览所有用户
- **权限**: 仅管理员
- **功能点**:
  - 用户列表分页展示
  - 显示用户基本信息、注册时间、文章数
  - 支持按用户名搜索

#### 3.6.2 增加用户
- **功能描述**: 创建新用户
- **权限**: 仅管理员
- **输入**: 用户名、密码、邮箱、角色

#### 3.6.3 编辑用户
- **功能描述**: 修改用户信息
- **权限**: 仅管理员
- **可修改字段**:
  - 邮箱
  - 角色
  - 状态（启用/禁用）

#### 3.6.4 删除用户
- **功能描述**: 删除用户
- **权限**: 仅管理员
- **业务规则**:
  - 不能删除自己
  - 软删除，保留用户文章归属

## 4. 非功能需求

### 4.1 性能需求
- 页面加载时间 < 2 秒
- API 响应时间 < 500ms（P95）
- 支持并发用户 100+

### 4.2 安全需求
- 密码使用 bcrypt 加密
- API 使用 JWT 认证
- 防止 SQL 注入（使用 ORM/参数化查询）
- XSS 防护
- 接口限流

### 4.3 可用性需求
- 系统可用性 99.9%
- 支持数据定期备份
- 错误日志记录

### 4.4 兼容性需求
- 支持主流浏览器（Chrome, Firefox, Safari, Edge）
- 移动端适配

---

## 5. 数据模型概览

### 5.1 核心实体
- **User**: 用户
- **Article**: 文章
- **Tag**: 标签
- **ArticleTag**: 文章-标签关联
- **Comment**: 评论
- **Attachment**: 附件
- **Category**: 分类
