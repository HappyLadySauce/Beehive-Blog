# 个人博客系统需求文档

## 1. 项目概述

### 1.1 项目名称
Beehive-Blog 个人博客系统

### 1.2 项目背景
本项目是一个基于 Go 语言开发的个人博客系统，旨在提供一个功能完善、易于管理的博客平台，支持文章发布、标签管理、评论互动、附件管理等核心功能。

### 1.3 技术栈
- **后端语言**: Go
- **数据库**: PostgreSQL
- **缓存**: Redis
- **API 文档**: Swagger
- **部署**: Docker

---

## 2. 用户角色定义

| 角色 | 角色标识 | 权限级别 | 说明 |
|------|----------|----------|------|
| 访客 | `guest` | 0 | 访客用户，可浏览内容和评论 |
| 普通用户 | `user` | 1 | 普通用户，可浏览内容和发表评论 |
| 管理员 | `admin` | 2 | 系统管理者，拥有全部权限 |

### 2.2 用户属性

| 属性名 | 类型 | 说明 |
|--------|------|------|
| `id` | bigint | 用户ID |
| `username` | string | 用户名 |
| `nickname` | string | 昵称 |
| `email` | string | 邮箱 |
| `password` | string | 加密密码 |
| `avatar` | string | 头像URL |
| `role` | enum | 角色：guest/user/admin |
| `status` | enum | 状态：active/inactive/disabled |
| `level` | int | 用户等级（1-6） |
| `experience` | int | 当前经验值 |
| `comment_count` | int | 评论数量 |
| `article_view_count` | int | 有效文章阅读数量 |
| `last_login_at` | datetime | 最后登录时间 |
| `created_at` | datetime | 注册时间 |
| `updated_at` | datetime | 更新时间 |

## 3. 功能需求

### 3.1 用户认证模块

#### 3.1.1 用户注册
- **功能描述**: 新用户注册账号
- **输入**: 用户名、密码、邮箱（可选）
- **输出**: 注册成功/失败信息
- **业务规则**:
  - 用户名必填，长度 3-20 字符
  - 密码长度 6-32 字符，需加密存储
  - 邮箱格式校验（如提供邮箱，需符合标准邮箱格式）
  - 需要邮箱验证，验证链接发送至用户邮箱，用户需点击链接激活账号

#### 3.1.2 用户登录
- **功能描述**: 用户身份验证
- **输入**: 用户名/邮箱、密码
- **输出**: JWT Token、用户信息
- **业务规则**:
  - 支持用户名或邮箱登录
  - 密码错误次数限制（5次/小时）
  - Token 有效期 7x24 小时

---

### 3.2 文章管理模块

#### 3.2.1 查看文章
- **功能描述**: 浏览文章列表和详情
- **权限**: 所有用户
- **功能点**:
  - 文章列表分页展示
  - 支持按分类、标签筛选
  - 支持关键字搜索
  - 文章详情展示（标题、内容、作者、发布时间、标签、评论数等）

#### 3.2.2 增加文章（管理员）
- **功能描述**: 发布新文章
- **权限**: 仅管理员
- **输入**: 标题、内容、分类（可选）、标签（可选）、封面图（可选）、状态
- **业务规则**:
  - 标题必填，长度 1-200 字符
  - 内容支持 Markdown 格式
  - 支持草稿和发布两种状态

#### 3.2.3 编辑文章（管理员）
- **功能描述**: 修改已有文章
- **权限**: 仅管理员
- **业务规则**:
  - 保留文章历史版本（可选）
  - 更新时间自动记录

#### 3.2.4 删除文章（管理员）
- **功能描述**: 删除文章
- **权限**: 仅管理员
- **业务规则**:
  - 软删除，保留数据可恢复
  - 删除时关联删除文章-标签关系

#### 3.2.5 文章分类管理（管理员）
- **功能描述**: 管理文章分类
- **权限**: 仅管理员
- **功能点**:
  - 创建分类（名称、描述、排序）
  - 编辑分类
  - 删除分类（需检查是否有关联文章）

#### 3.2.6 文章状态管理（管理员）
- **功能描述**: 设置文章状态
- **权限**: 仅管理员
- **状态定义**:
  - `draft`: 草稿
  - `published`: 已发布
  - `archived`: 已归档

---

### 3.3 标签管理模块

#### 3.3.1 查看标签
- **功能描述**: 浏览所有标签
- **权限**: 所有用户
- **功能点**:
  - 标签列表展示
  - 显示每个标签关联的文章数量
  - 点击标签查看关联文章

#### 3.3.2 增加标签（管理员）
- **功能描述**: 创建新标签
- **权限**: 仅管理员
- **输入**: 标签名称、颜色（可选）
- **业务规则**:
  - 标签名称唯一，长度 1-50 字符

#### 3.3.3 编辑标签（管理员）
- **功能描述**: 修改标签信息
- **权限**: 仅管理员

#### 3.3.4 删除标签（管理员）
- **功能描述**: 删除标签
- **权限**: 仅管理员
- **业务规则**:
  - 删除时自动解除与文章的关联关系

---

### 3.4 评论管理模块

#### 3.4.1 查看评论
- **功能描述**: 查看文章下的评论
- **权限**: 所有用户
- **功能点**:
  - 按文章查看评论列表
  - 支持评论分页
  - 支持评论回复（嵌套展示）

#### 3.4.2 发表评论
- **功能描述**: 对文章发表评论
- **权限**: 登录用户
- **输入**: 文章ID、评论内容、父评论ID（回复时）
- **业务规则**:
  - 评论内容长度 1-2000 字符
  - 支持回复其他评论
  - 新评论默认状态为 `pending`

#### 3.4.3 删除评论（管理员）
- **功能描述**: 删除不当评论
- **权限**: 仅管理员
- **业务规则**:
  - 软删除
  - 删除父评论时，子评论一并处理

#### 3.4.4 评论状态管理（管理员）
- **功能描述**: 审核评论状态
- **权限**: 仅管理员
- **状态定义**:
  - `pending`: 待审核
  - `approved`: 已通过
  - `rejected`: 已拒绝
  - `spam`: 垃圾评论

---

### 3.5 附件管理模块

#### 3.5.1 查看附件
- **功能描述**: 浏览上传的文件
- **权限**: 仅管理员
- **功能点**:
  - 附件列表展示

#### 3.5.2 上传附件
- **功能描述**: 上传图片/文件

- **支持类型**:

  - 其他: zip, rar, tar.gz
- **业务规则**:
  - 单文件最大 1GB
  - 自动生成缩略图（图片）
  - 文件重命名避免冲突

#### 3.5.3 删除附件
- **功能描述**: 删除附件
- **权限**: 仅管理员
- **业务规则**:
  - 物理删除文件
  - 删除数据库记录

#### 3.5.4 附件分类
- **功能描述**: 管理附件分类
- **权限**: 仅管理员
- **分类类型**:
  - `image`: 图片
  - `document`: 文档
  - `other`: 其他

## 4. 非功能需求

### 4.1 性能需求
- 页面加载时间 < 2 秒
- API 响应时间 < 500ms（P95）
- 支持并发用户 100+

### 4.2 安全需求
- 密码使用 bcrypt 加密
- API 使用 JWT 认证
- 防止 SQL 注入（使用 ORM/参数化查询）
- XSS 防护
- 接口限流

### 4.3 可用性需求
- 系统可用性 99.9%
- 支持数据定期备份
- 错误日志记录

### 4.4 兼容性需求
- 支持主流浏览器（Chrome, Firefox, Safari, Edge）
- 移动端适配

## 7. 用户等级系统

### 7.1 等级规则

| 等级 | 等级名称 | 所需经验值 | 升级条件 |
|------|----------|------------|----------|
| Lv.1 | 初来乍到 | 0 | 注册即获得 |
| Lv.2 | 博客新手 | 100 | 注册满7天 或 有效阅读10篇文章 |
| Lv.3 | 活跃读者 | 300 | 注册满30天 或 有效阅读30篇文章 |
| Lv.4 | 资深访客 | 600 | 注册满90天 或 有效阅读60篇文章 |
| Lv.5 | 博客达人 | 1000 | 注册满180天 或 有效阅读100篇文章 |
| Lv.6 | 资深博主 | 2000 | 注册满365天 或 有效阅读200篇文章 |

### 7.2 经验值获取规则

| 行为 | 经验值 | 限制条件 |
|------|--------|----------|
| 每日登录 | +5 | 每日限1次 |
| 有效阅读文章 | +10 | 单篇文章限1次，需停留≥3分钟 |
| 发表评论 | +15 | 每日限20条 |
| 文章被点赞 | +2 | 无限制 |
| 注册时间奖励 | 每天+1 | 按注册天数累计 |

### 7.3 有效阅读判定
- **判定条件**: 用户在文章页面停留时间 ≥ 3 分钟
- **防刷机制**: 
  - 同一用户同一文章24小时内只计算一次
  - 检测页面可见性（切换标签页不计时）
  - 检测用户行为（滚动、点击等交互）

## 8. 个人中心模块

### 8.1 我的信息
- **功能描述**: 查看和修改个人资料
- **权限**: 普通用户
- **可修改字段**:
  - 昵称
  - 头像
  - 个人简介
  - 邮箱
  - 密码

### 8.2 我的等级
- **功能描述**: 查看等级详情
- **展示内容**:
  - 当前等级和等级名称
  - 当前经验值
  - 升级进度条
  - 距离下一级所需经验
  - 等级特权说明

### 8.3 我的评论
- **功能描述**: 查看自己发表的所有评论
- **功能点**:
  - 评论列表（按时间倒序）
  - 跳转到对应文章
  - 删除自己的评论

### 8.4 阅读历史
- **功能描述**: 查看文章浏览记录
- **功能点**:
  - 浏览历史列表（按时间倒序）
  - 显示阅读时间和阅读时长
  - 清空历史记录
  - 删除单条记录

### 8.5 我的收藏
- **功能描述**: 查看收藏的文章
- **功能点**:
  - 收藏列表
  - 取消收藏
  - 按时间/分类筛选

## 9. 文章互动功能

### 9.1 文章点赞
- **功能描述**: 对文章进行点赞/取消点赞
- **权限**: 登录用户
- **业务规则**:
  - 同一用户同一文章只能点赞一次
  - 点赞数实时更新
  - 作者获得经验值奖励

### 9.2 文章收藏
- **功能描述**: 收藏感兴趣的文章
- **权限**: 登录用户
- **业务规则**:
  - 同一用户同一文章只能收藏一次
  - 可在个人中心查看收藏列表
